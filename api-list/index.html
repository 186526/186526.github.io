短链服务

API 请求地址：https://sl1.club/api/set.php
请求方式: POST

请求参数:

  • headers: Content-Type:application/json
  • POST参数内容: json
  • POST参数: {"url":url}

请求示范:

  • headers: Content-Type:application/json
  • POST内容: {"url":"186526.club"}
  • 返回参数:
    • 返回值: json
    • 原始值: {"success": "true","content": {"url": "http://sl1.club/807c9bc2"}}
    • 解析:
      • “success”:是否成功 返回布尔值
      • “content”: 若成功返回对象url 若失败返回原因（错误原因由UTF-8编码）
        • “url”: 返回缩短后链接

gravatar映像源

API 请求地址: https://mirrors.sl1.club/avatar
请求方法：get

请求参数：

  • /: 邮箱的md5值（32位小写）

请求示范：

  • 请求参数: https://mirrors.sl1.club/avatar/79af51057e869c392441146506ca2765
  • 返回参数：
    • 返回：jpeg（？）
